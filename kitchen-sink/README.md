# Kitchen Sink API

This folder contains a Siren API with containing all possible elements.

## Setup

```bash
# install dependencies
$ npm install

# setup env vars
$ cp .env.example .env
```

## Running the app

```bash
# development
$ npm start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm test

# watch mode
$ npm run test:watch

# test coverage
$ npm run test:cov

# e2e tests
$ npm run test:e2e
```
